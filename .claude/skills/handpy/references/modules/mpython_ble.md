# BLE 蓝牙 mpython_ble

`mpython_ble/` 预置于 v2/v3 用户文件系统，提供完整 BLE 功能。

```python
from mpython_ble.application.peripheral import Peripheral
from mpython_ble.application.centeral import Central
from mpython_ble.application.uart import UART
from mpython_ble.application.hid import HID
from mpython_ble.application.beacon import Beacon
```

官方文档（v2）：https://mPython.readthedocs.io
官方文档（v3）：https://mpython-esp32s3-doc.readthedocs.io/zh-cn/latest/
